The Urban Toolkit: A Grammar-based Framework for Urban Visual Analytics

About Urban Toolkit

The Urban Toolkit is a flexible and extensible visualization framework that enables the easy authoring of web-based visualizations through a new high-level grammar specifically built with common urban use cases in mind. This page presents 3 use cases that were built using the Urban Toolkit. It is possible to access each example by clicking on the links provided.

Feel free to modify the grammar and experiment with it :). Be aware that some join operations might take a few minutes to apply and load.

Due to the remote access some examples might take a couple of seconds to load.

Basic interactions

Some of the interactions need to be activated on the grammar

  • left click to move the camera
  • mouse wheel to zoom in and zoom out
  • shift + left click to rotate the camera
  • alt + right click to filter elements on the map
  • q undo filter of elements
  • t to select a building and embedded plots
  • alt + mouse wheel to change the height of a embedded plot
  • r to reset embedded plots and building selection

Footprint plots (Vis 1)

This example showcases the use of embedded plots. A radial plot was defined using vega-lite and can be applied on the buildings by pointing at it with the mouse and pressing the key t. It also possible to change the height of the plots by pointing at the building and using alt + mouse wheel. Test it here.

Linked plots (Vis 2)

In this example a histogram was created to show the distrbution of shadow in a section of manhattan, showcasing the use of linked plots. Also a knot was created to aggregated the shadow data per building. Test it here.

Operating knots (Vis 3)

This example illutrates the knot operations feature. A measure of sky exposure was calculated with and without trees. After loading both datasets and attaching them to the streets of Milan, it is possible to highlight their diferences by using the knot operations. Test it here.
